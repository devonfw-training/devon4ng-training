Basic Setup

  • Initialize a new Angular application with Angular CLI
  • Add @ngrx/store and @ngrx/store-devtools to the dependencies with npm/yarn
  • Download the Redux Devtools Extension

Configure @ngrx

  • Create root.reducer.ts in src/app
  • Define and export empty interface AppState (ignore warning - we add content later)
  • Define and export reducers - an empty object of type ActionReducerMap<AppState>
  • Add StoreModule and StoreDevtoolsModule to app.module and configure with root reducer

Check if configured successfully

  • inject Store<AppState> in AppComponent
  • serve application
  • check if redux devtools are initialized

Create login page

  • Create LoginPageComponent in src/app/auth
  • LoginPageComponent should be a simple form to log in a user

Define typed auth Actions I

  • create auth.actions.ts in src/app/auth

  • export two constants

    
export const LOGIN =  '[auth] Login';
export const LOGOUT = '[auth] Logout';

Define typed auth Actions II

  • export a class for each Action in auth.actions.ts


export class Login implements Action {
  readonly type = LOGIN;

  constructor(
    public payload: {
      user: {username: string, password: string}
    }
  ) {}
}

export class Logout implements Action {
    readonly type = LOGOUT;
}

Define typed auth Actions III

  • export a union type called AuthActions in auth.actions.ts

    
export type AuthActions = Login|Logout;

Create auth reducer

  • Create auth.reducer.ts in src/app/auth
  • Define AuthState and implement initialState and authReducer
  • AuthState defines the logged in user
  • Register authReducer in root.reducer reducers map

Implement auth reducer
