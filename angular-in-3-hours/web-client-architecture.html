Reducer Function


export function counterReducer(state: number = 0, action: Action) {
  switch (action.type) {
    case INCREMENT:
      return state + 1;

    case DECREMENT:
      return state - 1;

    case RESET:
      return 0;

    default:
      return state;
  }
}

... is a pure function that returns the new state for a given action.

With all that in mind - let's talk about the migration from Java Swing.

The most important thing: JavaScript/TypeScript has very little in common with Java.

While migrating, you can be happy to learn the coolest language there is. But be sure, there is no onboarding in 2 weeks - it will take time!

Migrating also means, changing the backend to HTTP resources and removing state and sessions.

Tooling is different in modern client development.

No need for heavyweights like Visual Studio or Eclipse.

Be ready to use just an editor with CLI tools.

Any web client is just a comfort function for the user to consume the backend.

Every backend resource needs to be secured. There is no trustworthy call from the client.
