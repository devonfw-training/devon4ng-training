TypeScript Exercises

Start by installing TypeScript


        npm install typescript --global
or 

        yarn global add typescript
If complete 
tsc --version
should return the installed version.

cd into /ts-exercises and run


        tsc -w exercises.ts --lib es2015

open /ts-exercises/result.html

... you will see failing UnitTests. Open /ts-exercises/exercises.ts and fix them.

With every file change, tsc will recompile and you can refresh your Browser to see the result - e.g. press F5.
