Http

The Angular Http client communicates with the server using a familiar HTTP request/response protocol.

The Http client is one of a family of services in the Angular HTTP library.

https://angular.io/docs/ts/latest/guide/server-communication.html

Http configuration

  • Importing HttpModule
  • Importing Http utils and Rxjs operators
  • Injecting Http service

Importing HttpModule


// import
import { HttpModule } from '@angular/http';

// module configuration
@NgModule({
    imports: [HttpModule],
    declarations: [...],
    bootstrap: [...],
    providers: [...]
})
export class AppModule {}

Importing Http utils and Rxjs operators


import { Http, Headers, RequestOptions, Response } from '@angular/http';
import { Observable } from 'rxjs/Observable';

import 'rxjs/add/operator/map';
import 'rxjs/add/operator/catch';

Injecting Http service


constructor(private http: Http) {}

Http GET


this.http.get(this.requestUrl + id)
      .map(res => res.json())

        console.log(error);
